Install Notes (ClawHub Package)

This bundle includes a minimal embedded zaomeng runtime for distribution.
It no longer depends on cloning an external repository at runtime.
Do not treat this file as the primary end-user workflow guide; agents should follow SKILL.md first.

Required files in this folder:
- SKILL.md
- runtime/zaomeng_cli.py
- runtime/src/...
- runtime/requirements.txt
- references/output_schema.md
- references/safety_policy.md
- references/validation_policy.md
- examples/sample_input_excerpt.txt
- examples/sample_character_profile.md
- examples/sample_relations.md
- examples/test-prompts.json

Embedded runtime notes:
- Required Python dependency: `PyYAML`
- Optional dependency: `tiktoken`
- Optional dependency for `.epub`: `ebooklib`
- On Windows / PowerShell, prefer `py -3 runtime/zaomeng_cli.py ...`
- Default runtime data path: `runtime/data/`
- Prompt templates and schemas in this bundle are packaging/reference files, not a substitute for the runtime entrypoint.

Quick validation checklist:
1) SKILL.md frontmatter has name + description.
2) `runtime/zaomeng_cli.py` can start.
3) Output keys strictly follow output_schema.md.
4) Safety rules are present and enforceable.
