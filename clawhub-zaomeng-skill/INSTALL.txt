Install Notes (ClawHub Package)

This bundle is a self-contained skill-definition package for distribution.
It is packaging metadata, not a replacement for the real local zaomeng engine.
Do not treat this file as runtime execution guidance for end-user tasks.

Required files in this folder:
- SKILL.md
- references/output_schema.md
- references/safety_policy.md
- references/validation_policy.md
- examples/sample_input_excerpt.txt
- examples/sample_character_profile.md
- examples/sample_relations.md
- examples/test-prompts.json

Optional local integration:
- If you already host zaomeng locally, map this skill to your local commands.
- Keep execution policy local-only by default.
- If the local zaomeng engine is absent, that means the real workflow is unavailable.
- Prompt templates and schemas in this bundle are for packaging/reference, not for silently simulating the engine in normal use.

Quick validation checklist:
1) SKILL.md frontmatter has name + description.
2) Output keys strictly follow output_schema.md.
3) Safety rules are present and enforceable.
