[Task]
Extract pairwise relationship graph from co-occurrence evidence.

[Input]
- chunk_text
- character_list

[Output]
- JSON object following references/output_schema.md (Relations Graph schema)

[Rules]
1. Only create relation keys for pairs that co-occur in the same chunk.
2. Use sorted key format: <A>_<B>.
3. Keep trust/affection in [0,10], power_gap in [-5,5].
4. Add concise typical_interaction and confidence.
5. If relation evidence is weak, keep neutral scores and low confidence.
