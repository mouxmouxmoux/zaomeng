[Task]
Distill character mind profiles from chunked novel text.

[Input]
- chunk_text
- character_candidates (optional)

[Output]
- Markdown character archive following references/output_schema.md (Character Profile schema)

[Rules]
1. Extract only evidence present in the text.
2. Keep values integer in [0,10].
3. Deduplicate core_traits/typical_lines/decision_rules.
4. If evidence is sparse, lower confidence and state why.
5. Never fabricate events not found in chunk_text.
